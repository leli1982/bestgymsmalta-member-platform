# BestGymsMalta Member App

Premium starter app for BestGymsMalta members.

## Features in this first version

- Premium landing page
- Member dashboard
- Check-in and branded workout story screen
- Rewards screen
- Gym passport screen
- Mobile-first PWA-style layout

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy on Vercel

1. Push this project to GitHub.
2. Open Vercel.
3. Import the GitHub repository.
4. Deploy.
